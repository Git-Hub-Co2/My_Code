# Timber
Tree cut game written in C with ncurses, inspired by "TreeDude" mini-game from "SuperHot". Game written in one afternoon.

## Dependencies

> * C compiler
> * Ncurses

```
$ gcc main.c -lncurses -o tree
$ ./tree
```
